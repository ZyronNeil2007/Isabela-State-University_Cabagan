# 🎓 Isabela State University Premium ID Maker

A premium, client-side web application for generating high-quality student identification cards for **Isabela State University (ISU)**. Featuring a state-of-the-art iOS 26-inspired specular glassmorphism interface, smooth 3D interactive animations, and a print-ready canvas renderer.

🔗 **Live Demo**: [https://zyronneil2007.github.io/Isabela-State-University_Cabagan/](https://zyronneil2007.github.io/Isabela-State-University_Cabagan/)

---

## ✨ Features

- **📱 Global Stepper Navigation**: Fully responsive step-by-step wizard layout for both desktop and mobile modes to save vertical space. Users enter one detail at a time, keeping the workspace incredibly clean and focused.
- **🔄 Smooth 3D Flip Card Animation**: Tap or click the main ID card to smoothly flip it between the front and back previews.
- **🖼️ Mini Live Card Preview (Flippable)**: A tiny, active thumbnail updates in real-time in the header. When you navigate past front-card details (Step 4) to back-card details (Step 5), the mini preview smoothly flips to match the side you are working on.
- **✍️ Digital Signature Pad**: A touch-and-mouse-friendly canvas allows you to sign right inside the browser and instantly places the signature on the ID template.
- **📷 Smart Photo Processor**: Drag, drop, or upload a portrait picture with automated canvas cropping and centering.
- **🖨️ High-Resolution A4 Print Export**: One-click print-ready PNG download. It automatically aligns both the front and back of the ID card on an A4 sheet with clean dashed cut-lines.
- **🫧 Premium Liquid Glass Aesthetics**: Styled using modern CSS3 blur backdrops, saturation, inner shadows, border gradients, and interactive hover feedback.

---

## 🛠️ Technology Stack

- **Structure**: Semantic HTML5
- **Styling**: Modern Vanilla CSS3 with Glassmorphism variables
- **Logic**: Vanilla ES6+ JavaScript (Canvas API, Touch Events, File Reader)
- **Icons**: Phosphor Icons (Web Integration)
- **Typography**: Inter & Roboto Condensed (Google Fonts)

## 🚀 Getting Started

No local servers, compilers, or extra setups are required. Everything is built using raw vanilla web technologies!

1. Clone or download the repository:
   ```bash
   git clone https://github.com/ZyronNeil2007/Isabela-State-University_Cabagan.git
   ```
2. Open `index.html` directly in any modern browser (Chrome, Safari, Edge, Firefox).

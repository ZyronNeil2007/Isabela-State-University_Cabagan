# 🎓 Isabela State University Premium ID Maker v2.0

A premium, fully responsive client-side web application for generating high-fidelity, print-ready student identification cards for **Isabela State University (ISU)**. Refactored from the ground up with a stunning, modern **Apple-style glassmorphism UI**, smooth 3D interactive preview components, and a custom high-resolution dual-card layout canvas renderer.

🔗 **Live Demo**: [https://zyronneil2007.github.io/Isabela-State-University_Cabagan/](https://zyronneil2007.github.io/Isabela-State-University_Cabagan/)

---

## ✨ Features

- **🫧 Premium Glassmorphism Aesthetics**: Styled with state-of-the-art CSS3 backdrop blurs, saturation layers, sleek gold (`#C9A84C`) & ISU green (`#15B915`) gradients, and tactile hover interactions.
- **📱 Fully Responsive Global Stepper**: A beautiful multi-step navigation wizard that organizes data inputs into logical, focused stages across both desktop and mobile layouts.
- **🔄 Interactive 3D Card Preview**: Experience immersive depth with real-time mouse-hover and touch 3D tilt effects Powered by VanillaTilt. Click or tap to flip the card seamlessly between its Front and Back views.
- **🖼️ Mini Live Card Preview**: A compact, interactive mini-thumbnail in the header updates in real-time and automatically flips to match the active steps you are working on.
- **✍️ Advanced Digital Signature Pad**: A touch-and-mouse-friendly HTML5 signature canvas that captures handwriting cleanly and embeds it dynamically on the ID template.
- **📷 High-Precision Photo Cropper**: Drop, drag, or upload a portrait picture with automatic canvas scaling, perfect cropping, and centered bounding grids.
- **🖨️ Professional A4 Print Renderer**: Instantly compile a print-ready A4 sheet containing both Front and Back sides of the ID card side-by-side, complete with clean dashed cut-lines for high-resolution physical printing.

---

## 🛠️ Technology Stack

- **Structure**: Semantic HTML5 with accessibility attributes (ARIA roles, keyboard-friendly navigation)
- **Styling**: Vanilla CSS3 using custom CSS properties (variables), high-performance GPU-accelerated transforms, and CSS grid/flexbox layouts
- **Logic**: Vanilla ES6+ JavaScript (Canvas API, Touch Events, FileReader, live visual syncing)
- **Interactions**: VanillaTilt.js (for the beautiful 3D parallax card hover effect)
- **Icons**: Phosphor Icons (Web Integration)
- **Typography**: Inter (Google Fonts)

## 🚀 Getting Started

No local servers, compilers, or dependencies are required. Everything is designed to run in pure vanilla web technologies:

1. Clone the repository:
   ```bash
   git clone https://github.com/ZyronNeil2007/Isabela-State-University_Cabagan.git
   ```
2. Double-click `index.html` or open it directly in any modern web browser.
